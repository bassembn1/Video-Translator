1. Install Python 3.8 or higher
2. Install FFmpeg (required for audio extraction):
   - **Windows**: Download from [ffmpeg.org](https://ffmpeg.org/download.html)
   - **macOS**: `brew install ffmpeg`
   - **Linux**: `sudo apt install ffmpeg`
